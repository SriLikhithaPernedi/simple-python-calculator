# 🧮 Simple Python Calculator

A beginner-friendly calculator project written in Python.  
This program allows users to perform basic arithmetic operations by entering two numbers and choosing an operation.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division (with zero-check)
- Exponentiation

## ▶️ How to Run
1. Make sure you have **Python 3** installed.
2. Clone or download this repository.
3. Run the script in your terminal:
   ```bash
   python (saved file name).py
